Apollo Puzzle

A classic block puzzle game for Android and iOS platforms. The game is built using native development on both platforms. The goal of the puzzles is moving any blocks so that the blue block can land inside the blue frame. Every puzzle is generated by a simulation based on a breath-first-search algorithm and some heuristic functions. Each one of them is guaranteed to have a solution.
